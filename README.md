# The-best-song-Spotify-2025

# วัตถุประสงค์
- วิเคราะห์ข้อมูล Spotify จาก Kaggle เพื่อค้นหาเพลงที่ดีที่สุดของปี2025 และ ทำ insight หาว่าเพลงที่ดังหรือเพลงที่ทำส่วนใหญ่จะมีเนื้อหาที่หยาบคาย(Explicit)ไหม?

# Key insight
- เพลงที่เป็น Explicit มีค่า ความนิยมเฉลี่ยสูงกว่า เพลงที่ไม่ Explicit สรุปได้ว่าในปี2025 เพลงที่มีเนื้อหาที่หยาบคาบ/ตรงไปตรงมา มีแนวโน้มจะได้รับความนิยมมากกว่า 
- แม้ว่าเพลงที่มีเนื้อหา Explicit จะมีค่าเฉลี่ยความนิยมสูงกว่า แต่เพลงที่ติดอันดับความนิยมสูงสุด(15อันดับ)ส่วนใหญ่เป็นเพลงที่ไม่มีเนื้อหา Explicit สะท้อนให้เห็นว่าความนิยมของเพลงไม่ได้ขึ้นอยู่กับการเป็น Explicit เพียงอย่างเดียว แต่ยังได้รับอิทธิพลจากปัจจัยอื่น เช่น ชื่อเสียงของศิลปิน กระแสในโซเชียลมีเดีย แนวเพลง และความสามารถในการเข้าถึงผู้ฟังในวงกว้าง
